对象更改检测注意事项

由于javascript的限制，Vue不能检测对象属性的添加或者删除。

对于已经创建的实例，Vue不能动态添加根级别的响应式属性。但是，可以使用Vue.set(object, key, value)方法向嵌套对象添加响应式属性。例如，对于：

      
        var vm = new Vue({
          data: {
            userProfile: {
              name: 'Anika'
            }
          }
        })

可以添加一个新的age属性到嵌套的userProfile对象：

      
        Vue.set(vm.userProfile, 'age', 27)

可以使用vm.$set实例方法，这是全局Vue.set的别名：

      
        vm.$set(vm.userProfile, 'age', 27)

有时候需要为已有对象赋予多个新属性，在这种情况下应该使用以下方法：

      
        vm.userProfile = Object.assign({}, vm.userProfile, {age: 27, favoriteColor: 'Green'})
